## ğŸš€ Getting Started

### Prerequisites

- A modern web browser.
- A Google Gemini API Key (set via environment variables in production).

### Development

The project is structured as a standalone React application using ESM modules for maximum performance and zero-bundler overhead in the browser.

1. Clone the repository:
   ```bash
   git clone https://github.com/Quit4859/AcminX.git
   ```
2. Serve the root directory using any local web server (e.g., `npx serve .`).
3. Ensure your `process.env.API_KEY` is configured to access the generation features.
